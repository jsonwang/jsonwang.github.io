macOS-Sierra编译安装libimobiledevice

发表于 2017-02-11   |     |   阅读次数

一,libimobiledevice 能做什么

  • 打包生成好的测试ipa 直接安装集成到设备.
  • 无线查看iOS 设备 安装的APP,LOG,crash信息,共享目录等.
  • 可让iPhone进行重启,关机,睡眠等.

二,编译安装

1,安装libimobiledevice 确认下载依赖库openssl、 libplist 、libusb、libusbmuxd
–>
2,安装ideviceinstaller

三,安装ifuse

五,部分命令使用说明,所有的命令都支持 -u XXXX device UDID 具体看命令的 help

  • idevice_id 查看设备
    idevice_id -l //输出所有已联接设备
  • idevicedate 显示,修改设备的时间
  • ideviceenterrecovery
  • idevicename 显示设备用户名
  • idevicescreenshot 截屏并保存到当前目录XXX.tiff
  • idevicebackup 这个命令是iOS 3 or below 已经不work 要使用 idevicebackup2 命令
  • idevicebackup2 设备的备份和恢复等操作
  • idevicedebug
  • ideviceimagemounter
  • idevicenotificationproxy
  • idevicesyslog 显示设备的log
  • idevicedebugserverproxy
  • ideviceinfo 设备的信息(手机号,WIFI地址, 设备颜色,哪些APP 是重复安装等) 未完
  • idevicepair
  • idevicecrashreport 设备的CrashLOG 信息操作
  • idevicediagnostics
  • ideviceinstaller
  • ideviceprovision
