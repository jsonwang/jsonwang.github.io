Xcode8.2 自定义模板

发表于 2017-02-08   |   分类于 杂谈   |     |   阅读次数

一,自定义模板好处

  • 快速创建工程目录结构
  • 节省重复代码手写时间
  • 统一规范代码，提高代码可读性

二,工程模板位置

  • 里面包含文件模板（File Templates）和工程模板（Project Templates）

  • MacOSX的系统文件模板位置

  • 用户自定义模板位置,我们自定义的都要放在这个目录下才可以生效

三,如何使用

  • 1,下载工程模版文件 https://github.com/jsonwang/NX-Xcode-Template
  • 2,复制文件到 ~/Library/Developer/Xcode/Templates
