MAC 硬盘清理重点

发表于 2016-04-30   |   分类于 杂谈   |     |   阅读次数

「其他」部分占用磁盘空间往往不小，那么到底那些文件组成了「其他」部分呢？
OS X 可以识别应用程序、备份、音频、电影、备份、和照片等集中文件类型，而「其他」部分则含括了除上述提到的文件类型的其他文件类型。这意味着，「其他」部分包含了相当多的文件类型。常见的如下列表：

  • 文件和文件类型，包括.pdf、.doc、.psd 等文件类型
  • 存档文件和磁盘映像，包括 .zip、.dmg、.iso 等文件类型
  • 个人文件和用户数据（user data）
  • OS X 系统文件夹内的文件，如系统的临时文件等
  • 用户库文件，如应用程序支持文档、iCloud 文档、屏保程序等
  • 用户缓存和系统缓存，比如浏览器缓存和存储在本地的对话信息
  • 字体文件、应用程序插件、应用程序扩展等
  • Spotlight 无法识别的文件类型，如虚拟机的磁盘驱动器，Windows Boot Camp 启动分区等


我的电脑是 MacBook Pro（Retina 显示屏，13 英寸，2013 年末）闪存是120GB 很不够用,其中 硬盘中的其它占很大部分,我今天清理了 75GB左右的空间,这下爽多了.
我的电脑环境:
xcode Version 7.3 (7D175)

OS X EI 10.11.4


一定要注意在删除清理文件时一定要确认功能有些操作是不可逆的做好备份

具体清理目录

  • 1.删除 xcode 设备支持的老版本和不用的设备支持

  • 2.手机系统日志

  • 3.手机系统日志

  • 4.删除DerivedData

  • 5.删除QuickLook缓存文件 慎用

  • 5.删除缓存文件

  • 6.删除不用的模拟器版本

  • 7.删除文字转语音功能OS X内置的嗓音文件

  • 8.移除模拟器中安装的Apps 直接重置模拟器

  • 9.移除Archives 建议备份dSYM文件夹 慎用

  • 10.移除旧的Docsets

  • 11.最容易被忽视的我们项目的.svn 文件时间越长越大 提交所有代码在重新co 一份新代码,git 同理

  • 12.删除不使用的桌面背景图片 440M

  • 13.屏幕保护图片这个小点 147M左右 看个人情况是否删除

  • 14.还有两个大文件 swapfile0 sleepimage 不建议删除 慎用

推荐两个软件

  • OmniDiskSweeper找出硬盘中的大文件
  • CleanMyMac 清理硬盘但文章中说的文件不会被删除

整理不是全部的,同学们有别的大的文件可以给我留言我整理上去.
